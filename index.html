หน่วยที่ 2 — สารละลาย
ครอบคลุมมัธยมต้น → มัธยมปลาย · 4 บท · เรียงตามลำดับการเรียนรู้
เลือกบทด้านล่างเพื่ออ่านรายละเอียด
1
องค์ประกอบของสารละลายและปัจจัยที่มีผลต่อสภาพละลายได้
ระดับมัธยมต้น — พื้นฐานสำคัญก่อนเรียนทุกบท
ส่วนประกอบ
ตัวละลาย + ตัวทำละลาย
สารละลายคือของผสมเนื้อเดียวกัน (homogeneous mixture) มี 2 ส่วน
  • ตัวทำละลาย (solvent) — ปริมาณมากกว่า กำหนดสถานะ เช่น น้ำ, แอลกอฮอล์
  • ตัวละลาย (solute) — ปริมาณน้อยกว่า ถูกละลาย เช่น เกลือ, น้ำตาล, CO₂
เข้าใจง่าย
ลองนึกภาพชงน้ำหวาน — น้ำคือ "ตัวทำละลาย" (มีเยอะกว่า) ส่วนผงน้ำหวานคือ "ตัวละลาย" ที่ถูกน้ำดูดซับเข้าไป จนกลายเป็นของเหลวใสเนื้อเดียวกัน
น้ำเกลือ น้ำหวาน น้ำโซดา ทองเหลือง
ระดับการอิ่มตัว
สภาพละลายได้ของสาร
  • ไม่อิ่มตัว — ยังละลายเพิ่มได้
  • อิ่มตัว — ละลายได้สูงสุดที่ T นั้น
  • อิ่มตัวยวดยิ่ง — เกินจุดอิ่มตัว ไม่เสถียร
สภาพละลายได้ = มวลตัวละลายสูงสุด (g)
ต่อตัวทำละลาย 100 g ณ อุณหภูมิหนึ่ง
เข้าใจง่าย
เหมือนใส่น้ำตาลลงในชาร้อน — ช่วงแรกละลายหมด (ไม่อิ่มตัว) ใส่ไปเรื่อย ๆ จนน้ำตาลเริ่มตกตะกอน นั่นคือจุด "อิ่มตัว" แล้ว
ปัจจัยที่มีผลต่อสภาพละลายได้
อุณหภูมิ
Temperature
คายความร้อน ΔH < 0
T↑ → ละลายได้↓
ละลายดีที่ T ต่ำ
NaOH, HCl, Ca(OH)₂
ดูดความร้อน ΔH > 0
T↑ → ละลายได้↑
ละลายดีที่ T สูง
KNO₃, NH₄NO₃
เข้าใจง่าย
คายความร้อน → เหมือนแช่ก๊าซในน้ำเย็น ยิ่งเย็นยิ่งอยู่นิ่ง ยิ่งร้อนก็หนีออก เช่น น้ำอัดลมอุ่น ๆ จะไม่ซ่าเพราะก๊าซหนีออกหมด

ดูดความร้อน → เหมือนชงน้ำตาล ยิ่งใช้น้ำร้อนยิ่งละลายได้เร็วและมากขึ้น
หลัก Le Chatelier: T↑ → ระบบต้านด้วยปฏิกิริยาดูดความร้อน ∴ สารดูดความร้อนละลายได้มากขึ้นเมื่อร้อน สารคายความร้อนละลายได้น้อยลง
ความดัน
Pressure
มีผลเฉพาะก๊าซที่ละลายในของเหลว
ความดัน↑ → สภาพละลายได้↑
กฎ Henry: S = k · P
เข้าใจง่าย
เหมือนน้ำอัดลมในขวด — โรงงานอัดก๊าซ CO₂ ใส่ไปด้วยความดันสูงจึงซ่า พอเปิดฝา "ฟู่" คือความดันลดลงทันที ก๊าซเลยออกมา
ธรรมชาติสาร
Like dissolves like
สารขั้วละลายในตัวทำละลายขั้ว
สารไม่มีขั้วละลายในตัวทำละลายไม่มีขั้ว
เข้าใจง่าย
"เพื่อนสนิทละลายด้วยกัน" — เกลือ (มีขั้ว) ละลายน้ำ (มีขั้ว) ได้ดี แต่ไขมัน (ไม่มีขั้ว) ไม่ละลายน้ำ นั่นเหตุว่าทำไมน้ำมันลอยแยกชั้น
NaCl + น้ำ ✓ ไขมัน + เฮกเซน ✓ ไขมัน + น้ำ ✗
2
ความเข้มข้นของสารละลาย
ม.ต้น: ร้อยละ 3 แบบ → ม.ปลาย: โมลาริตี โมแลลิตี เศษส่วนโมล
ระดับมัธยมต้น — ร้อยละ
%w/w
ร้อยละโดยมวล
= (มวลตัวละลาย / มวลสารละลาย) × 100
เข้าใจง่าย
ใส่เกลือ 5 g ลงในน้ำ 95 g รวมมวลสารละลาย = 100 g → ความเข้มข้น 5% นั่นคือทุก 100 กรัม มีเกลืออยู่ 5 กรัม
%v/v
ร้อยละโดยปริมาตร
= (ปริมาตรตัวละลาย / ปริมาตรสารละลาย) × 100
เข้าใจง่าย
แอลกอฮอล์ล้างแผล 70% หมายความว่า ในทุก ๆ 100 mL มีเอทานอลจริง ๆ อยู่ 70 mL ที่เหลือเป็นน้ำ
%w/v
ร้อยละมวล/ปริมาตร
= (มวลตัวละลาย g / ปริมาตร mL) × 100
เข้าใจง่าย
น้ำเกลือ 0.9%w/v ที่โรงพยาบาล = ละลายเกลือ 0.9 กรัมในน้ำแล้วปรับปริมาตรให้ได้ 100 mL พอดี
ระดับมัธยมปลาย — หน่วยโมล
Molarity
โมลาริตี (M)
M = mol ตัวละลาย / L สารละลาย
เข้าใจง่าย
คิดแบบ "กี่โมลต่อขวด 1 ลิตร" — NaOH 1M คือขวด 1 ลิตรที่มี NaOH 1 โมล (40 g) ละลายอยู่
Molality
โมแลลิตี (m)
m = mol ตัวละลาย / kg ตัวทำละลาย
เข้าใจง่าย
คล้าย Molarity แต่ใช้ "น้ำหนักน้ำ" แทน "ปริมาตรสารละลาย" เพราะน้ำหนักไม่เปลี่ยนตามอุณหภูมิ จึงแม่นยำกว่าในการคำนวณ Colligative
Mole fraction
เศษส่วนโมล (X)
Xₐ = nₐ / (nₐ + n_B + …)
เข้าใจง่าย
ถ้ามีน้ำ 9 โมล + เกลือ 1 โมล → X_เกลือ = 1/10 = 0.1 แปลว่า "1 ใน 10 โมเลกุลทั้งหมด เป็นเกลือ"
3
กระบวนการละลายและพลังงาน
ระดับมัธยมปลาย — ΔH, ΔS, ΔG และหลัก Le Chatelier
3 ขั้นตอนพลังงาน
วัฏจักรการละลาย (ไอออนิก)
  1. แยกพันธะผลึก → พลังงานโครงผลึก (+, ดูดความร้อน)
  2. แยกโมเลกุลน้ำ → พลังงานแยกตัวทำละลาย (+, ดูด)
  3. ไอออน + น้ำ → พลังงานไฮเดรชัน (−, คายความร้อน)
ΔH_soln = ΔH_lattice + ΔH_hydration
เข้าใจง่าย
ลองนึกภาพทำลายกำแพง 2 ด้าน (ขั้น 1–2 ต้องใช้พลังงาน) แล้วปล่อยให้สองฝ่ายเข้าหากัน (ขั้น 3 คายพลังงานออกมา) ผลรวมสุดท้ายคือ ΔH_soln ถ้าบวก = ดูดรวม, ถ้าลบ = คายรวม
ΔH < 0 — คาย
NaOH, H₂SO₄ → ร้อน
ถุงประคบร้อน
ΔH > 0 — ดูด
NH₄NO₃ → เย็น
ถุงประคบเย็น
เทอร์โมไดนามิกส์
ทิศทางการละลาย — ΔG
การละลายเกิดขึ้นเองเมื่อ ΔG < 0 โดยขึ้นกับ 2 ปัจจัย
ΔH (เอนทาลปี) — พลังงานที่เปลี่ยนแปลง
+
TΔS (เอนโทรปี × T) — ความไม่เป็นระเบียบ ↑ ช่วยให้ละลาย
ΔG = ΔH − TΔS < 0 → ละลายเอง
เข้าใจง่าย
เอนโทรปีคือ "ความต้องการแตกกระจาย" ของโมเลกุล — เหมือนน้ำหอมที่เปิดฝาขวดแล้วกระจายเองทั่วห้อง ความปรารถนาที่จะกระจายนี้ช่วยผลักดันการละลาย แม้จะต้องใช้พลังงาน (ΔH > 0)
4
คุณสมบัติ Colligative ของสารละลาย
ระดับมัธยมปลาย — ขึ้นกับจำนวนอนุภาค ไม่ขึ้นกับชนิดสาร
เข้าใจง่าย — ภาพรวม
Colligative properties คือสมบัติที่ขึ้นกับ "จำนวน" อนุภาคตัวละลาย ไม่ว่าจะเป็นสารอะไร — เหมือนจำนวนคนในลิฟต์ที่กดน้ำหนัก ไม่สนว่าเป็นใคร ยิ่งใส่ตัวละลายมาก ผลก็ยิ่งชัด
1 — Vapor pressure lowering
การลดลงของความดันไอ
P_soln = X_solvent · P°_solvent
ΔP = X_solute · P°_solvent
เข้าใจง่าย
โมเลกุลตัวละลายนั่ง "ปิดทาง" ผิวน้ำ ทำให้โมเลกุลน้ำหนีออกเป็นไอได้น้อยลง → ความดันไอลดลง ยิ่งใส่ตัวละลายมาก ยิ่งระเหยยาก
ตัวอย่างในชีวิตประจำวัน
น้ำเกลือ/น้ำเชื่อมระเหยช้ากว่าน้ำบริสุทธิ์ — อาหารหมักดองเก็บได้นานขึ้น
น้ำทะเลระเหยช้ากว่าน้ำจืด เพราะมีเกลือแร่ละลายอยู่มาก ส่งผลต่อวัฏจักรน้ำ
อุตสาหกรรมยาผสมตัวทำละลายลดความดันไอ ป้องกันส่วนผสมระเหยง่ายสูญหาย
2 — Boiling point elevation
การเพิ่มของจุดเดือด (ΔT_b)
ΔT_b = K_b · m · i
K_b น้ำ = 0.512 °C·kg/mol | NaCl → i = 2 | น้ำตาล → i = 1
เข้าใจง่าย
เพราะตัวละลายลดความดันไอ น้ำเลยต้องร้อนกว่า 100°C กว่าจะเดือด — เหมือนต้องออกแรงมากขึ้นเพื่อข้ามกำแพง
ตัวอย่างในชีวิตประจำวัน
ต้มมะกะโรนีใส่เกลือน้ำมีจุดเดือดสูงขึ้นเล็กน้อย อาหารสุกเร็วขึ้น
สารหล่อเย็นรถยนต์ (coolant)ผสม ethylene glycol ป้องกันน้ำในหม้อน้ำเดือดเร็วเกินไป
ทำลูกอม/น้ำเชื่อมเข้มข้นต้มน้ำตาลเข้มข้นที่อุณหภูมิสูงกว่า 100°C เพื่อเนื้อสัมผัสตามต้องการ
3 — Freezing point depression
การลดลงของจุดเยือกแข็ง (ΔT_f)
ΔT_f = K_f · m · i
K_f น้ำ = 1.86 °C·kg/mol | จุดเยือกแข็ง = 0°C − ΔT_f
เข้าใจง่าย
ตัวละลาย "รบกวน" การจัดเรียงผลึกน้ำแข็ง ทำให้น้ำต้องเย็นกว่า 0°C กว่าจะแข็ง — เหมือนคนแปลกหน้าที่เข้ามาขัดขวางการรวมกลุ่ม
ตัวอย่างในชีวิตประจำวัน
โรยเกลือถนนหิมะNaCl ลดจุดเยือกแข็งต่ำกว่า 0°C ป้องกันน้ำแข็งก่อตัวบนพื้นถนน
น้ำยา antifreezeผสม ethylene glycol ป้องกันน้ำแข็งตัวในหม้อน้ำรถช่วงฤดูหนาว
ทำไอศกรีมแบบโบราณแช่ภาชนะในน้ำแข็งผสมเกลือ เย็นต่ำกว่า 0°C จึงแข็งตัวได้
เลือดและน้ำเลี้ยงในร่างกายมีจุดเยือกแข็งต่ำกว่า 0°C ป้องกันเซลล์แข็งตัวในอากาศเย็น
4 — Osmotic pressure
ออสโมซิสและความดันออสโมติก (π)
π = MRT
M = โมลาริตี | R = 0.08206 L·atm/mol·K | T ในหน่วย K
เข้าใจง่าย
น้ำวิ่งจากฝั่งเจือบาง → ฝั่งเข้มข้น เพื่อ "เจือจาง" ฝั่งที่เข้มข้นกว่า เหมือนฝูงชนกระจายออกจากที่แออัด ความดันออสโมติก (π) คือแรงที่ต้องอัดกลับเพื่อหยุดการไหลนั้น
ตัวอย่างในชีวิตประจำวัน
เครื่องกรองน้ำ ROอัดความดันสูงกว่า π ดันน้ำบริสุทธิ์ผ่านเมมเบรน กรองเกลือออก
ผักเหี่ยวเมื่อโรยเกลือเกลือดึงน้ำออกจากเซลล์ผัก (hypertonic) ทำให้เซลล์เสียน้ำและเหี่ยว
น้ำเกลือ 0.9% ในโรงพยาบาลเป็น isotonic กับเลือด ให้สารน้ำทางหลอดเลือดโดยไม่ทำให้เม็ดเลือดแตก
รากต้นไม้ดูดน้ำจากดินน้ำเลี้ยงในรากเข้มข้นกว่าน้ำในดิน น้ำซึมเข้ารากตามออสโมซิสโดยไม่ใช้พลังงาน
isotonic = ความดันเท่ากัน · hypertonic → เซลล์เหี่ยว (plasmolysis) · hypotonic → เซลล์บวม/แตก (lysis)
ตัวประกอบ van't Hoff (i)
เข้าใจง่าย
i คือ "จำนวนอนุภาคที่ได้จริงเมื่อละลาย" — น้ำตาลไม่แตกตัวเลย (i=1) แต่ NaCl แตกเป็น 2 ไอออน (i=2) ดังนั้นเกลือมีผลต่อ Colligative มากกว่าน้ำตาลที่ความเข้มข้นเท่ากัน
น้ำตาล/กลูโคส i = 1
NaCl → Na⁺ + Cl⁻ i = 2
CaCl₂ → Ca²⁺ + 2Cl⁻ i = 3
Fe(NO₃)₃ → Fe³⁺ + 3NO₃⁻ i = 4
เลือกบทจากปุ่มด้านบน · กล่องสีเขียว = คำอธิบายภาษาง่าย · กล่อง mono = สูตรคำนวณจริง
